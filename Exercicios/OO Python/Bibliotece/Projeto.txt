Projeto: Biblioteca de Livros

O objetivo deste projeto é criar uma classe Livro para representar um livro e uma classe Biblioteca para gerenciar uma coleção de livros.

A classe Livro deve ter as seguintes propriedades:

    título: string
    autor: string
    isbn: string
    disponível: booleano

E os seguintes métodos:

    init(self, título, autor, isbn): construtor que recebe os valores de título, autor e ISBN e define a propriedade disponível como True
    emprestar(self): método que define a propriedade disponível como False se o livro estiver disponível, caso contrário, exibe uma mensagem informando que o livro não está disponível
    devolver(self): método que define a propriedade disponível como True se o livro estiver emprestado, caso contrário, exibe uma mensagem informando que o livro não precisa ser devolvido

A classe Biblioteca deve ter as seguintes propriedades:

    livros: lista de objetos Livro

E os seguintes métodos:

    init(self): construtor que inicializa a lista de livros vazia
    adicionar_livro(self, livro): método que recebe um objeto Livro e adiciona à lista de livros
    emprestar_livro(self, título): método que recebe o título de um livro, procura na lista de livros e chama o método emprestar do objeto Livro correspondente, caso o livro esteja disponível
    devolver_livro(self, título): método que recebe o título de um livro, procura na lista de livros e chama o método devolver do objeto Livro correspondente, caso o livro esteja emprestado
